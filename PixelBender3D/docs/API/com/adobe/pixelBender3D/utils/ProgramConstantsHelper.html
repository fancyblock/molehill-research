Packagecom.adobe.pixelBender3D.utils
Classpublic final class ProgramConstantsHelper
InheritanceProgramConstantsHelper Object

Manage passing parameters to the runtime - make it easy to pass in parameters of different types. Create one instance of this class per vertex / fragment program combination, set all of the variables you need, then call update before drawing.
